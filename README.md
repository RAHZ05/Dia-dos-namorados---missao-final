## 🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando as tecnologias base da web:

- **HTML5**: Estrutura semântica do projeto.
- **CSS3**: Estilização, layout responsivo e animações fluidas.
- **JavaScript (ES6+)**: Lógica do jogo, manipulação de estados e eventos de interação.

## 🕹️ Sobre o jogo
O objetivo do jogo é capturar corações que caem na tela antes do tempo acabar. 

O jogador interage através do movimento do mouse / as  teclas do teclado <img width="45" height="30" alt="tteclas-removebg-preview" src="https://github.com/user-attachments/assets/9f1552a6-30b2-4649-811d-a590f4ba95fd" />

## 💡 O que aprendi
Este projeto foi uma excelente oportunidade para aprofundar meus conhecimentos em:
- **Manipulação do DOM**: Atualização dinâmica da interface sem recarregar a página.
- **Lógica de Programação**: Gestão de estados, controle de tempo e eventos de usuário.
- **Animações**: Uso de CSS para tornar a interface mais envolvente.
